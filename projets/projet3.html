Matete

Application mobile et site web de ventes de fruits et légumes

C'est quoi?

Matete est une plateforme destinée à la vente de légumes entre particuliers via des annonces. Elle propose un site web Symfony ainsi qu'une application mobile dévelopée sur Android Studio reliée au serveur mySQL via une API en PHP. L'integration continue (tests unitaires et revues de code) est mise en place pour ce projet, ainsi que le deploiement continu, tout cela via Gitlab.
