Projet personnel

Size Matters

Size Matters est un site web d'une complexité notable. Le back-office permet de récupérer chaque jours les informations d'influenceurs ciblés sur Instagram. Le site présente des rapports précis avec des graphiques et des statistiques. Le projet est développé avec Ruby on Rails et comprend beaucoup de librairies JavaScript. Le système de récupération des données sur Instagram est développé en Python.
