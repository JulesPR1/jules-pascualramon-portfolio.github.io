IP Warm-Up

Application web de Warm-Up d'adresse IP

C'est quoi?

Cette application web développée avec le framework Ruby on Rails est une application visant à faire "chauffer" des adresses IP afin d'améliorer leurs réputations aux yeux des differents services de messageries mail. C'est le projet de mon 1er stage de BTS, d'une durée de 5 semaines.
