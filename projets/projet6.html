Talk-Crypto

Forum et statistique de cryptomonnaies

C'est quoi?

Talk-Crypto est un forum sur le thème de la cryptomonnaie où tout le monde est libre de poster des articles. Le site propose également une option permettant d'afficher des informations (graphique des prix, description) sur une cryptomonnaie choisie. Le site est fait avec le framework Ruby on Rails et le framework chartJS pour les graphiques. Les données sur les cryptomonnaies proviennent de l'API CoinGeko. Le projet existe également sous forme d'application de bureau pour Ubuntu, faite avec le framework Electron. Ceci est un projet personnel réalisé dans le but d'auto-évaluer et augmenter mes compétences.
